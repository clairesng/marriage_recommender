# PNG Archetype Asset Brief

Use 7 custom illustrated PNG files in this folder with these exact names:

- power_couple.png
- island_tribe.png
- dynamic_explorers.png
- practical_architects.png
- fire_ice_duo.png
- safety_net_couple.png
- balanced_builders.png

## Art Direction

- Style: flat vector-like character illustration, soft pastel palette, clean outlines, playful but warm
- Format: PNG with transparent background
- Suggested size: 1400x1400 px (square)
- Composition: central couple focus, readable at smaller UI sizes
- Keep visual language consistent across all 7 assets

## Scene Suggestions Per Archetype

- `power_couple.png`: couple confidently collaborating (e.g., cooking/planning together)
- `island_tribe.png`: couple united while external family/social elements are in background
- `dynamic_explorers.png`: energetic, playful couple in motion/adventure pose
- `practical_architects.png`: couple doing structured activity (planning, budgeting, reading together)
- `fire_ice_duo.png`: contrasting personalities in one frame but still connected
- `safety_net_couple.png`: supportive pose (one comforting/grounding the other)
- `balanced_builders.png`: calm, everyday partnership scene with equal participation

## Current App Behavior

The app now loads PNG first. If a PNG is missing, it falls back to the existing SVG for that archetype.
